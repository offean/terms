Política de Privacidade

Última atualização: 29 de junho de 2026

Esta Política de Privacidade descreve como as informações são tratadas durante a utilização deste bot no Discord.

Dados Coletados

O bot não coleta dados pessoais sensíveis, como nome completo, CPF, endereço, telefone, e-mail, localização ou qualquer outra informação privada dos usuários.

Para que suas funcionalidades operem corretamente, o bot poderá armazenar informações técnicas disponibilizadas pela API do Discord, incluindo, mas não se limitando a:

  • ID de usuários;
  • ID de servidores (Guild ID);
  • ID de canais;
  • ID de cargos;
  • ID de mensagens, quando necessário para funcionalidades específicas;
  • ID do proprietário do servidor;
  • Quantidade de membros do servidor;
  • Configurações definidas pelos administradores do servidor;
  • Outros identificadores públicos fornecidos pela API do Discord necessários ao funcionamento do bot.

Finalidade

Essas informações são utilizadas exclusivamente para permitir o funcionamento do bot, armazenar configurações dos servidores, executar comandos e oferecer recursos personalizados.

Compartilhamento

As informações armazenadas não são vendidas, alugadas ou compartilhadas com terceiros, exceto quando houver obrigação legal.

Segurança

São adotadas medidas razoáveis para proteger as informações armazenadas contra acessos não autorizados, alterações indevidas ou perda de dados.

Retenção dos Dados

As informações poderão permanecer armazenadas enquanto forem necessárias para o funcionamento do bot ou até que sejam removidas pelo usuário, pelo administrador do servidor ou pelo desenvolvedor.

Alterações desta Política

Esta Política de Privacidade poderá ser atualizada periodicamente para refletir alterações no funcionamento do bot ou em requisitos legais.

Contato

Caso tenha dúvidas sobre esta Política de Privacidade ou deseje solicitar informações relacionadas aos dados armazenados, entre em contato através do servidor oficial de suporte do bot.
