Política de Privacidade

Esta Política de Privacidade descreve como as informações são tratadas durante a utilização deste bot no Discord.

1. Dados Coletados

O bot não coleta dados pessoais sensíveis, como nome completo, endereço, documentos, e-mail, telefone, localização, senhas ou qualquer informação privada dos usuários.

Para que determinadas funcionalidades funcionem corretamente, o bot poderá armazenar algumas informações disponibilizadas pela própria API do Discord, incluindo, mas não se limitando a:

  • ID de usuários;
  • ID de servidores (Guild ID);
  • ID de canais;
  • ID de cargos;
  • ID de mensagens;
  • ID do proprietário do servidor;
  • Quantidade de membros do servidor;
  • Configurações definidas pelos administradores do servidor;
  • Outras informações públicas disponibilizadas pelo Discord necessárias para o funcionamento do bot.

Essas informações são consideradas dados técnicos e públicos dentro da plataforma Discord e são utilizadas exclusivamente para o correto funcionamento das funcionalidades oferecidas pelo bot.

2. Finalidade

Os dados armazenados são utilizados apenas para:

  • Executar comandos;
  • Salvar configurações dos servidores;
  • Permitir recursos personalizados;
  • Melhorar a estabilidade e funcionamento do bot.

3. Compartilhamento

O bot não vende, aluga ou compartilha quaisquer informações armazenadas com terceiros, salvo quando exigido por lei.

4. Segurança

São adotadas medidas razoáveis para proteger os dados armazenados contra acessos não autorizados, alterações indevidas ou perda das informações.

5. Retenção dos Dados

As informações poderão permanecer armazenadas enquanto forem necessárias para o funcionamento do bot ou até que sejam removidas mediante solicitação ou exclusão das funcionalidades correspondentes.

6. Alterações desta Política

Esta Política de Privacidade poderá ser modificada a qualquer momento para refletir melhorias, novas funcionalidades ou alterações legais.

7. Contato

Em caso de dúvidas sobre esta Política de Privacidade, entre em contato com a equipe responsável pelo bot através do servidor oficial de suporte.
